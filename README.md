# sure_tomorrow_insurance_ml
Este es el repositorio en el que se creará todo para el proyecto del sprint 14 del curso de Triple Ten 
# Sure Tomorrow Insurance Machine Learning Project

## Descripción del proyecto
Este proyecto se centra en el análisis y modelado de datos de clientes de la compañía de seguros **Sure Tomorrow** para resolver cuatro tareas clave utilizando técnicas de machine learning:

1. **Clientes similares**: Identificar clientes que sean similares a un cliente determinado para mejorar estrategias de marketing.
2. **Predicción de beneficios de seguro (clasificación)**: Predecir si un nuevo cliente recibirá un beneficio de seguro utilizando un modelo de predicción, comparándolo con un modelo dummy.
3. **Predicción de cantidad de beneficios (regresión)**: Estimar cuántos beneficios de seguro probablemente recibirá un nuevo cliente usando regresión lineal.
4. **Protección de datos personales**: Desarrollar un algoritmo de transformación de datos que dificulte la recuperación de información personal sin afectar la calidad de los modelos.

## Dataset
El dataset utilizado se encuentra en el archivo `insurance_us.csv` y contiene las siguientes columnas:

- `sexo`: Género del cliente asegurado.
- `edad`: Edad del cliente.
- `salario`: Salario anual del cliente.
- `numero_familiares`: Número de familiares del cliente asegurado.
- `beneficios`: Número de beneficios de seguro recibidos en los últimos cinco años (objetivo).

## Instrucciones de uso
1. Cargar los datos desde `insurance_us.csv`.
2. Verificar la integridad y limpieza de los datos.
3. Realizar cada una de las tareas descritas en el proyecto.
4. Evaluar los modelos y extraer conclusiones basadas en los resultados.

## Objetivo
El proyecto busca demostrar la aplicabilidad de técnicas de machine learning en la identificación de patrones de clientes y predicción de beneficios de seguros, así como implementar estrategias de protección de datos sensibles.
